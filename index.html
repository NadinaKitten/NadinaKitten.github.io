Luau - Roblox Studio

My First Adventure (tentatively named)

A casual game spanning several places within one experience. The game mechanics aren't fully decided on yet, as I am really using this experience to learn Luau and familiarize myself with game development software. After realizing I wanted to share this experience with others and make it both enjoyable and secure, I decided to use ProfileService, developed by MadStudio.

I have successfully created a quest module and tutorial quest line, and received help from the devforum to make my code simpler. I am currently reworking the places to include teleports between different areas. The plan is to have a tutorial zone, where players can learn everything about every aspect of the game, and dedicated areas for specific skills such as farming, forestry, gathering, and mining. Teleportation is working properly thanks to this video, and has given me the confidence needed to break this large project into smaller, more managable pieces.

Also on my list of development is implementing a level system for each of the life skill trees, with perks for leveling both the skills themselves, and the tools related(where tools are required to perform actions). The most ambitious piece by far is the customization - I want to make my game accessible, which to me requires allowing players to change color settings for the overall GUI, as well as the NPC interactions. I was inspired by this post and continue to keep it in mind while developing other apps outside of Roblox.

This adventure is slated to be complete by the end of 2024.

Python

Data analysis, on the surface, might look boring to some, but to me it is exciting. Even at work, with my limited access to reports, I am always excited by the possibliies of how you can retrieve large amounts of data quickly, and what knowledge you can glean by analyzing it. Thus, my journey with Python began.

Using the learning path on LinkedIn Learning, I followed a series of courses which introduced me to Python, and culminated in created a Quiz App. This ran entirely in the terminal, and while it worked, it wasn't exactly a work of art, or very accessible. Currently, I am using Tkinter to design a GUI for the app, and have successfully created the main page, allowing the user to enter their name before proceeding to the main menu.

This project is taking me a lot of time to complete, mainly because my understanding of the limitations of Tkinter was incomplete. I have tested it, and determined that the way I had originally wanted things to look would not be possible, and had to change my approach drastically. Also, aside from actually taking quizzes, I am implementing ways to allow the user to customize their experience, from changing the colors and fonts to allowing the option to display a timer while taking quizzes or keeping it hidden.

Completion of the QuizApp is set for December 2023. After the quiz app is complete, my next challenge is to create a Solitaire game.
